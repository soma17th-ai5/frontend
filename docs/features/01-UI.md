# [기능명] AI 챗봇 기본 UI/UX 및 글로벌 레이아웃 세팅

## 1. 개요 (Objective)

- SomaAgent의 메인 인터페이스인 대화형 챗봇 화면의 기본 뼈대(Layout)와 디자인 시스템을 구축한다.
- Figma 디자인(AI-Chatbot-UI-UX-Design)을 기반으로 사용자(User)와 AI(Agent) 간의 채팅 UI 요소들을 원자(Atomic) 단위 컴포넌트로 분리하여 구현한다.

## 2. 기술 스택 (Tech Stack)

- Next.js 16 (App Router)
- Tailwind CSS (스타일링 및 반응형 구현)
- Lucide React (아이콘)
- (선택) Framer Motion (부드러운 메시지 등장 애니메이션 및 UX 개선)

## 3. UI/UX 요구사항 (Requirements)

### 3.1. 글로벌 레이아웃 (Global Layout)

- [ ] **사이드바 (Sidebar):** 좌측에 위치하며, 이전 대화 내역(History)과 새 대화 시작 버튼 포함. 모바일에서는 햄버거 메뉴로 숨김 처리.
- [ ] **메인 영역 (Main Area):** 우측(또는 중앙)에 위치하며, 헤더, 채팅 로그 영역, 하단 입력창으로 구성.

### 3.2. 챗봇 핵심 컴포넌트 (Chat Components)

- [ ] **ChatMessage (말풍선):**
  - **사용자(User):** 우측 정렬, 브랜드 컬러(또는 어두운 색상) 배경.
  - **에이전트(Agent):** 좌측 정렬, 밝은 회색 또는 흰색 배경. SomaAgent 프로필 아이콘 표기.
- [ ] **SourceChip (출처 표시):** AI 답변 하단에 노출되는 뱃지 형태. 클릭 시 해당 링크(OpenSoma 공지 등)로 이동.
- [ ] **ActionButtons (액션 제안 버튼):** AI가 특정 행동(예: "신청하기", "취소하기")을 제안할 때 누를 수 있는 버튼 UI.

### 3.3. 입력창 (Chat Input Area)

- [ ] **Textarea:** 사용자가 길게 입력할 수 있도록 글자 수에 따라 높이가 자동 조절(Auto-resize)되는 텍스트 영역.
- [ ] **버튼:** 전송(Send) 버튼 및 첨부파일/도구(툴) 호출 아이콘 배치.
- [ ] 빈 값일 때는 전송 버튼 비활성화 (Disabled 상태 처리).

### 3.4. UX 인터랙션 (UX Interactions)

- [ ] **Scroll to Bottom:** 새로운 메시지가 추가되거나 AI가 답변을 생성 중일 때, 자동으로 스크롤이 가장 아래로 부드럽게 이동(Smooth Scroll)해야 함.
- [ ] **Loading State:** AI가 답변을 생성하는 동안 사용자에게 진행 중임을 알리는 로딩 인디케이터(예: 타이핑 애니메이션 쩜쩜쩜 `...` 또는 스켈레톤 UI) 노출.

## 4. 폴더 구조 가이드 (Component Structure)

Cursor는 다음 구조에 맞춰 컴포넌트를 생성할 것:

- `src/components/layout/Sidebar.tsx`
- `src/components/chat/ChatContainer.tsx`
- `src/components/chat/ChatMessage.tsx`
- `src/components/chat/ChatInput.tsx`
- `src/components/ui/` (버튼, 칩 등 재사용 가능한 공통 컴포넌트)
